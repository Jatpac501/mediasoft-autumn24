# Контактный cписок

**Фамилия Имя**: _Улитин Евгений_  
**Тестовое задание на JS**

## Описание проекта

Это веб-приложение для управления контактами. Оно позволяет пользователям добавлять, редактировать, удалять и помечать контакты как избранные.
Контакты сохраняются в cookies браузера, чтобы данные оставались доступными после перезагрузки страницы.
Если контакт не имеет фотографии, будет использоваться случайный цвет фона с первой буквой имени.

### Основные функции:

- **Добавление контактов**: с фото или с случайным фоном и первой буквой имени
- **Удаление контактов**: возможность удаления контакта
- **Избранные контакты**: отметить контакт как избранный
- **Поиск по контактам**: фильтрация по имени или номеру телефона
- **Cookies**: сохранение данных контактов в cookies без фотографий

## Подготовительные действия

1. Убедиться, что установлен браузер
2. Убедиться, что JavaScript включен в браузере

## Информация о доступах

Логин/Пароль не требуется. Все данные хранятся локально в cookies.
